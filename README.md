# API Marketplace

Бэкенд для платформы, где пользователи могут выставлять на продажу цифровые товары.

## Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Скопируйте `.env.example` в `.env` и при необходимости заполните переменные.
3. Запустите сервер в режиме разработки:
   ```bash
   npm run dev
   ```
